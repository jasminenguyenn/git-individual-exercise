# Git Individual Exercise

This repo is for my individual Git exercise.

I made a small Java program that shows two simple features.

Feature 1:
- Sorting a list of integers in ascending order using `Collections.sort`.

Feature 2:
- Sorting a list of strings in alphabetical order using `Collections.sort`.

## How to run

1. Open the project in your Java IDE (I used IntelliJ).
2. Make sure `Main.java` is inside the `src` folder.
3. Run the `Main` class.
4. You will see the original lists and the sorted lists printed in the console.

## Tests

For this exercise, my “tests” are the console outputs:
- I check that the integers are printed unsorted first and then in ascending order.
- I check that the strings are printed unsorted first and then in alphabetical order.
